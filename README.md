# 🔍 AEO Diagnostic Tool

**Built for Pixii.ai Founding Engineer Project**

See how your Amazon product ranks across 3 leading AI engines simultaneously.

## 🚀 What it does

Paste any shopper query (e.g. "best magnesium supplement for seniors") and your brand name — the tool queries 3 AI engines at once and gives you:

- ✅ Visibility score for each AI engine
- ✅ Where your brand ranks in AI recommendations
- ✅ Overall AEO Report Card
- ✅ Actionable recommendations to improve your ranking

## 🛠️ Built With

- Python
- Streamlit
- Groq API (Llama 4 Scout + Llama 3.3 Versatile + Llama 3.1 8b)

## ▶️ How to Run

1. Clone this repo
2. Install dependencies:
```bash
pip install streamlit google-generativeai groq
```
3. Add your Groq API key in app.py
4. Run:
```bash
streamlit run app.py
```
